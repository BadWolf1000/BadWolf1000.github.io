Process Commands

  • ps -ef Running processes $ ps -ef | head -3
    UID PID PPID CMD
    root 1 0 /sbin/init
  • ps aux All processes detailed $ ps aux | grep ssh
    root 892 0.0 sshd: /usr/sbin/sshd
  • top Live processes $ top
    PID USER %CPU %MEM COMMAND
    892 kali 12.3 4.1 firefox
  • htop Interactive process viewer $ htop
    (colorized, scrollable process list)
  • kill PID Kill process $ kill 1234
    (process 1234 terminated)
  • kill -9 PID Force kill process $ kill -9 1234
    (process forcefully killed)
  • pkill name Kill by name $ pkill firefox
    (all firefox processes killed)
  • bg Resume in background $ bg
    [1]+ nmap 10.10.10.0/24 &
  • fg Resume in foreground $ fg
    nmap 10.10.10.0/24
  • jobs List background jobs $ jobs
    [1]+ Running nmap 10.10.10.0/24 &
  • nohup command & Run immune to hangup $ nohup ./scan.sh &
    nohup: output to 'nohup.out'
  • nice -n 10 command Run with priority $ nice -n 19 hashcat -m 0 hash.txt wordlist
    (runs with lowest priority)
  • lsof -i :80 Who uses port 80 $ lsof -i :80
    apache2 1234 root TCP *:80 (LISTEN)
  • strace -p PID Trace system calls $ strace -p 1234
    read(3, "data...", 4096) = 128
