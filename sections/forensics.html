  • exiftool image.jpg View metadata $ exiftool photo.jpg

  • exiftool -all= image.jpg Strip metadata $ exiftool -all= photo.jpg

  • binwalk file Analyze binary $ binwalk firmware.bin

    0x4000 Zip archive
  • binwalk -e file Extract embedded files $ binwalk -e firmware.bin
