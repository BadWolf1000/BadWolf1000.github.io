Important Shortcuts

  • Ctrl + C Stop/terminate $ ping google.com
    ^C (stops the running ping)
  • Ctrl + Z Pause process $ vim file.txt
    ^Z [1]+ Stopped vim file.txt
  • Ctrl + D Exit/logout $ python3
    >>> ^D (exits interpreter)
  • Ctrl + L Clear screen (clears terminal, same as clear)
  • Ctrl + R Reverse search history (reverse-i-search)`nmap': nmap -sV 10.10.10.5
  • Ctrl + A Move to line start (cursor jumps to beginning of line)
  • Ctrl + E Move to line end (cursor jumps to end of line)
  • Ctrl + U Delete to line start (clears text before cursor)
  • Ctrl + W Delete previous word $ nmap -sV 10.10| ^W
    $ nmap -sV |
  • Tab Auto-complete $ cat /etc/pa[TAB]
    $ cat /etc/passwd
  • !! Repeat last command $ apt update
    $ sudo !!
    sudo apt update
  • !$ Last argument $ cat /etc/hosts
    $ vim !$
    vim /etc/hosts
  • clear Clear screen $ clear
    (terminal screen cleared)
