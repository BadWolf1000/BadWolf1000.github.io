Archive & Compression

  • tar -czf file.tar.gz dir Create gzip archive $ tar -czf backup.tar.gz /home/kali
    (archive created)
  • tar -xzf file.tar.gz Extract gzip archive $ tar -xzf backup.tar.gz
    (files extracted to current dir)
  • tar -cjf file.tar.bz2 dir Create bzip2 archive $ tar -cjf archive.tar.bz2 project/
    (bzip2 compressed archive created)
  • tar -xjf file.tar.bz2 Extract bzip2 archive $ tar -xjf archive.tar.bz2
    (bzip2 archive extracted)
  • tar -tf file.tar.gz List archive contents $ tar -tf backup.tar.gz
    home/kali/notes.txt
    home/kali/scripts/
  • tar -xzf file.tar.gz -C /path Extract to specific dir $ tar -xzf backup.tar.gz -C /tmp
    (extracted to /tmp)
  • zip -r archive.zip dir/ Zip directory recursively $ zip -r project.zip project/
    adding: project/main.c (deflated 58%)
  • zip test.zip file.txt Create zip $ zip loot.zip data.csv
    adding: data.csv (deflated 62%)
  • unzip test.zip Extract zip $ unzip loot.zip
    inflating: data.csv
  • unzip -l archive.zip List zip contents $ unzip -l loot.zip
    Length Name
    2048 data.csv
  • gzip file.txt Compress single file $ gzip largefile.log
    $ ls
    largefile.log.gz
  • gunzip file.txt.gz Decompress gzip file $ gunzip largefile.log.gz
    $ ls
    largefile.log
  • xz file.txt XZ compress $ xz database.sql
    $ ls
    database.sql.xz
  • unxz file.txt.xz XZ decompress $ unxz database.sql.xz
    $ ls
    database.sql
  • 7z x archive.7z Extract 7zip archive $ 7z x backup.7z
    Extracting archive: backup.7z
    Everything is Ok
