Advanced Scanning

  • nmap -sS target SYN stealth scan $ sudo nmap -sS 10.10.10.5
    22/tcp open ssh (SYN scan)
  • nmap -sU target UDP scan $ sudo nmap -sU 10.10.10.5
    53/udp open domain
    161/udp open snmp
  • nmap -sN target TCP null scan $ sudo nmap -sN 10.10.10.5
    (no flags set, evades some firewalls)
  • nmap -sA target ACK scan (firewall test) $ sudo nmap -sA 10.10.10.5
    80/tcp unfiltered http
    443/tcp filtered https
  • nmap -A target Aggressive scan $ nmap -A 10.10.10.5
    OS: Linux 5.x | Services + scripts
  • nmap -T4 target Faster timing $ nmap -T4 -p- 10.10.10.5
    (aggressive timing, faster scan)
  • nmap --min-rate 5000 target Set minimum packet rate $ nmap --min-rate 5000 -p- 10.10.10.5
    (fast full port scan)
  • masscan -p1-65535 target Fast port scan $ masscan -p1-65535 10.10.10.5 --rate=1000
    open tcp 22, 80, 443
  • nikto -h target Web vuln scan $ nikto -h http://10.10.10.5
    + /admin/: Directory indexing found
  • enum4linux target SMB enumeration $ enum4linux 10.10.10.5
    | Share: public Type: Disk
  • snmpwalk -v2c -c public target SNMP enumeration $ snmpwalk -v2c -c public 10.10.10.5
    sysDescr.0 = Linux target 5.15.0
  • nmap --script smb-enum-shares target SMB share scan $ nmap --script smb-enum-shares 10.10.10.5
    | public: READ
    | admin$: NO ACCESS
  • rustscan -a target Ultra-fast port scan $ rustscan -a 10.10.10.5
    Open 10.10.10.5:22,80,443
    (scans all 65535 ports in seconds)
