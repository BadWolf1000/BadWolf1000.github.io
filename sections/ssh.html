SSH & Remote

  • ssh user@server Connect to server $ ssh admin@192.168.1.10
    admin@server:~$
  • ssh -p 2222 user@server Connect on custom port $ ssh -p 2222 kali@10.10.10.5
    kali@target:~$
  • ssh -i key.pem user@server Connect with private key $ ssh -i ~/.ssh/id_rsa admin@10.10.10.5
    admin@target:~$
  • scp file user@server: Copy to server $ scp exploit.py kali@10.10.10.5:/tmp
    exploit.py 100% 2KB
  • scp user@server:file local Copy from server $ scp kali@10.10.10.5:/etc/passwd ./loot/
    passwd 100% 1.2KB
  • scp -r dir user@server: Copy directory to server $ scp -r tools/ kali@10.10.10.5:/tmp
    linpeas.sh 100% 620KB
  • ssh-keygen Generate SSH key $ ssh-keygen -t rsa -b 4096
    Generating public/private rsa key pair.
  • ssh-copy-id user@server Copy key to server $ ssh-copy-id kali@10.10.10.5
    Number of keys added: 1
  • ssh user@server 'command' Run remote command $ ssh admin@10.10.10.5 'cat /etc/hostname'
    target-server
  • sftp user@server Interactive file transfer $ sftp kali@10.10.10.5
    sftp> get /etc/shadow
    Fetching /etc/shadow
  • ssh -N -f -L port:host:port user@server Background tunnel $ ssh -N -f -L 3306:localhost:3306 admin@10.10.10.5
    (background MySQL tunnel)
  • eval $(ssh-agent) && ssh-add key SSH agent forwarding $ eval $(ssh-agent) && ssh-add ~/.ssh/id_rsa
    Identity added: /home/kali/.ssh/id_rsa
