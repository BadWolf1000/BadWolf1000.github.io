Must Know

  • sudo Run as admin $ sudo apt update
    [sudo] password for kali: ****
  • sudo su Switch to root $ sudo su
    root@kali:/home/kali#
  • vi file.txt Edit file $ vi config.txt
    (press i to insert, :wq to save)
  • nano file.txt Simple editor $ nano notes.txt
    (^X to exit, ^O to save)
  • echo "hello" Print text $ echo "Hello, World!"
    Hello, World!
  • grep pattern file Search in file $ grep "root" /etc/passwd
    root:x:0:0:root:/root:/bin/bash
  • man command Show manual $ man ls
    LS(1) - list directory contents
  • history Command history $ history
    1 ls -la
    2 cd /tmp
    3 whoami
  • alias ll='ls -la' Create alias $ alias ll='ls -la'
    $ ll
    total 32 drwxr-xr-x ...
  • export VAR=value Set environment var $ export PATH=$PATH:/opt/tools
    $ echo $PATH
    /usr/bin:/opt/tools
  • env Show environment $ env
    HOME=/home/kali
    PATH=/usr/local/bin:...
  • xargs Build commands from input $ cat urls.txt | xargs -I {} curl {}
    (curls each URL in file)
  • watch -n 5 command Repeat every 5 seconds $ watch -n 5 'netstat -tuln'
    (refreshes every 5s)
