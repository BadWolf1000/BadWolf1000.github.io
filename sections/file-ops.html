File Operations

  • touch file.txt Create file $ touch notes.txt
    $ ls
    notes.txt
  • cat file.txt Show file content $ cat /etc/hostname
    kali
  • head -n 20 file.txt First 20 lines $ head -n 5 /etc/passwd
    root:x:0:0:root:/root:/bin/bash
    ...
  • tail -n 20 file.txt Last 20 lines $ tail -n 3 /var/log/auth.log
    Jan 28 sshd: Accepted publickey
  • tail -f log.txt Live logs $ tail -f /var/log/syslog
    Jan 28 12:00:01 kali CRON[1234]: ...
  • less file.txt Scrollable viewer $ less /var/log/syslog
    (use q to quit, / to search)
  • wc -l file.txt Count lines $ wc -l /etc/passwd
    42 /etc/passwd
  • mv old.txt new.txt Rename/move file $ mv report.txt final_report.txt
  • cp file.txt copy.txt Copy file $ cp config.yml config_backup.yml
  • cp -r dir/ backup/ Copy directory $ cp -r project/ project_bak/
  • rm file.txt Delete file $ rm temp.txt
    $ ls
    (file removed)
  • ln -s target link Create symlink $ ln -s /opt/tools/nmap /usr/local/bin/nmap
  • file document Detect file type $ file mystery
    mystery: ELF 64-bit LSB executable
  • diff file1 file2 Compare files $ diff old.conf new.conf
    3c3
    < port=80
    > port=443
  • tee file.txt Write stdin to file $ echo "log" | tee output.txt
    log
