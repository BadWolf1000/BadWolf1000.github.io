SMB & Windows

  • smbclient -L //target List SMB shares $ smbclient -L //10.10.10.5 -N
    Sharename Type Comment
    public Disk Public files
  • smbclient //target/share Connect to SMB share $ smbclient //10.10.10.5/public -N
    smb: \> ls
    backup.zip 4096
  • smbclient //target/share -U user Authenticated SMB $ smbclient //10.10.10.5/admin -U admin%Pass123
    smb: \> get flag.txt
  • smbmap -H target SMB share enum $ smbmap -H 10.10.10.5
    [+] public READ
    [+] admin$ NO ACCESS
  • smbmap -H target -u user -p pass Authenticated share enum $ smbmap -H 10.10.10.5 -u admin -p Pass123
