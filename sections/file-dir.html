File & Directory Commands

  • pwd Show current directory $ pwd
    /home/kali/Desktop
  • ls List files $ ls
    file.txt notes scripts
  • ls -l Detailed list $ ls -l
    -rw-r--r-- 1 kali kali 420 Jan 1 file.txt
  • ls -a Show hidden files $ ls -a
    . .. .bashrc .profile file.txt
  • ls -lh Human readable sizes $ ls -lh
    -rw-r--r-- 1 kali kali 4.2K Jan 1 file.txt
  • ls -R Recursive list $ ls -R
    ./src:
    main.c utils.c
    ./lib:
    libfoo.so
  • cd foldername Change directory $ cd /var/log
    $ pwd
    /var/log
  • cd .. Go back one folder $ cd ..
    $ pwd
    /var
  • cd ~ Go to home directory $ cd ~
    $ pwd
    /home/kali
  • cd - Go to previous directory $ cd -
    /var/log
  • mkdir test Create directory $ mkdir project
    $ ls
    project
  • mkdir -p a/b/c Create nested dirs $ mkdir -p src/lib/utils
    (creates all parent dirs)
  • rmdir folder Remove empty directory $ rmdir old_project
    (removed if empty)
  • rm -rf folder Force remove directory $ rm -rf temp/
    (recursively deleted)
  • tree Visual directory tree $ tree
