Permissions

  • chmod 755 script.sh Set execute permission $ chmod 755 deploy.sh
    $ ls -l deploy.sh
    -rwxr-xr-x 1 kali kali 512 ...
  • chmod +x script.sh Make executable $ chmod +x scan.sh
    $ ./scan.sh
    Running scan...
  • chmod 600 id_rsa Owner read/write only $ chmod 600 ~/.ssh/id_rsa
    (only owner can read/write)
  • chmod -R 777 dir/ Recursive full access $ chmod -R 777 /tmp/share/
    (all permissions for everyone)
  • chown user file Change owner $ sudo chown www-data index.html
  • chown user:group file Change owner and group $ sudo chown www-data:www-data /var/www/html
  • chgrp group file Change group only $ chgrp developers project/
    (group set to developers)
  • umask 022 Set default permissions $ umask 022
    (new files: 644, dirs: 755)
  • ls -l See permissions $ ls -l
    -rw-r--r-- 1 root root 1024 ...
  • stat file Detailed file info $ stat file.txt
    Access: (0644/-rw-r--r--) Uid: 1000
  • getfacl file Show ACL permissions $ getfacl secret.txt
    user::rw- group::r-- other::---
  • setfacl -m u:user:rwx file Set ACL for user $ setfacl -m u:kali:rwx shared.txt
