Network

  • ip a Show IP address $ ip a
    eth0: inet 192.168.1.105/24
  • ifconfig Network interfaces $ ifconfig eth0
    inet 192.168.1.105 netmask 255.255.255.0
  • ip route Show routing table $ ip route
    default via 192.168.1.1 dev eth0
  • ping google.com Test connectivity $ ping -c 2 google.com
    64 bytes from 142.250.80.46: ttl=118
  • traceroute target Trace packet route $ traceroute 10.10.10.5
    1 192.168.1.1 1ms
    2 10.0.0.1 5ms
  • curl url Fetch web content $ curl -I http://target
    HTTP/1.1 200 OK
  • netstat -tuln Show open ports $ netstat -tuln
    tcp 0.0.0.0:22 LISTEN
    tcp 0.0.0.0:80 LISTEN
  • ss -tuln Socket statistics $ ss -tuln
    LISTEN 0 128 0.0.0.0:22 0.0.0.0:*
  • arp -a ARP table $ arp -a
    gateway (192.168.1.1) at aa:bb:cc:dd:ee:ff
  • nslookup domain.com DNS query $ nslookup google.com
    Address: 142.250.80.46
  • tcpdump -i eth0 Capture packets $ sudo tcpdump -i eth0 port 80
    IP 192.168.1.105 > 10.10.10.5: HTTP
  • iptables -L List firewall rules $ sudo iptables -L
    Chain INPUT (policy ACCEPT)
    Chain FORWARD (policy DROP)
  • nmcli dev status Network manager status $ nmcli dev status
    eth0 ethernet connected Wired
