Tunneling & Pivoting

  • ssh -L 8080:target:80 user@pivot Local port forward $ ssh -L 8080:172.16.0.5:80 user@10.10.10.5
    (localhost:8080 → internal:80)
  • ssh -R 8080:localhost:80 user@pivot Remote port forward $ ssh -R 8080:localhost:80 user@10.10.14.5
    (remote:8080 → your local:80)
  • ssh -D 9050 user@pivot SOCKS proxy $ ssh -D 9050 user@10.10.10.5
    (SOCKS proxy on localhost:9050)
  • chisel server -p 8000 --reverse Chisel server $ chisel server -p 8000 --reverse
    server: Listening on 0.0.0.0:8000
  • socat TCP-LISTEN:8080,fork TCP:target:80 Port forwarding $ socat TCP-LISTEN:8080,fork TCP:172.16.0.5:80
    (forwards 8080 → internal:80)
