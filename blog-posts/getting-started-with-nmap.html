Getting Started with Nmap: A Beginner's Guide

Nmap (Network Mapper) is one of the most powerful and widely-used tools in the cybersecurity world. Whether you're a system administrator checking your network's security or a penetration tester on an engagement, understanding Nmap is essential.

What is Nmap?

Nmap is a free, open-source tool for network discovery and security auditing. It uses raw IP packets to determine what hosts are available on the network, what services those hosts are offering, what operating systems they're running, and dozens of other characteristics.

Basic Host Discovery

The simplest way to use Nmap is to scan a single host:

nmap 192.168.1.1

This performs a basic scan of the most common 1000 ports on the target.

Scanning Multiple Hosts

You can scan an entire subnet:

nmap 192.168.1.0/24

Or specify a range:

nmap 192.168.1.1-50

Common Scan Types

  • -sS - TCP SYN scan (stealth scan)
  • -sT - TCP connect scan
  • -sU - UDP scan
  • -sV - Version detection
  • -O - OS detection

Service and Version Detection

To identify what services are running and their versions:

nmap -sV 192.168.1.1

This is crucial for identifying potentially vulnerable software versions.

Aggressive Scan

For a comprehensive scan that includes OS detection, version detection, script scanning, and traceroute:

nmap -A 192.168.1.1
Always ensure you have proper authorization before scanning any network. Unauthorized scanning can be illegal and unethical.

Saving Output

Save your results for later analysis:

nmap -oN output.txt 192.168.1.1    # Normal output
nmap -oX output.xml 192.168.1.1    # XML output
nmap -oG output.gnmap 192.168.1.1  # Grepable output

Conclusion

Nmap is an incredibly versatile tool with many more features than covered here. As you become more comfortable, explore NSE scripts, timing options, and advanced evasion techniques. Check out the Linux Commands section of this site for a quick reference!
